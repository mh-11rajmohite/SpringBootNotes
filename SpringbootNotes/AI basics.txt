

#Interpreted and Compliled languages:

*Interpreter language : 
	-No need to generate machine instructions(oject code)
	-Has its own virtual enviroment that communiactes with te computer
	-Machine independent
	-Has its own virtual environment to communicates with computer.
	-ex. Pyhton,Perl,PHP,Ruby
	-Language->ReadyToRun->VM->MachineCode

*Compiled lang. : 
	-Code is converted to object code
	-Change in code requires recompilation
	-Manual efforts required to run code	
	-sx. C,C++,Rust,Go
	-Language->MachineCode->ReadyToRun

4.Then it will asked to procced : type - 'Y'
